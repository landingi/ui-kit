Vertical Alignment

```jsx
  <Legend data= {
    [{ range: '0 - 19', variant: 'success' },
    { range: '20 - 59', variant: 'warning' },
    { range: '60 - 100', variant: 'alert' }]
  } />
```
  
Horizontal Alignment
```jsx
  <Legend alignment='horizontal' data= {
    [{ range: '0 - 19', variant: 'success' },
    { range: '20 - 59', variant: 'warning' },
    { range: '60 - 100', variant: 'alert' }]
  } />
```
