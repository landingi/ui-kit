Default Input
```jsx
<IntlProvider>
  <Input  />
</IntlProvider>
```
Input with placeholder
```jsx
<IntlProvider>
  <Input placeholder='placeholder' />
</IntlProvider>
```
Read Only Input
```jsx
<IntlProvider>
  <Input readonly={true} />
</IntlProvider>
```
Disabled Input
```jsx
<IntlProvider>
  <Input  disabled={true} />
</IntlProvider>
```
Input with Label
```jsx
<IntlProvider>
  <Input  label='label' />
</IntlProvider>
```
Input name
```jsx
<IntlProvider>
  <Input  name='inputName' />
</IntlProvider>
```
